In this repository, I intend to gather tutorials for the ones that like to know more about computer vision 
libraries.

List of contents:

1. Simple image Operations:
   - `cv2.imread`
   - `cv2.imwrite`
   - `cv2.cvtColor`
   - `cv2.resize`
   - `cv2.split`
   - `cv2.merge`
   - `cv2.imshow`
   - `cv2.waitKey`
   - `cv2.bitwise_not`
   - `cv2.bitwise_and`
   - `cv2.bitwise_or`

2. Simple Video Operation:
   - `cv2.VideoCapture`
   - `cv2.VideoWriter`
   - `cv2.VideoWriter_fourcc`

3. Drawing on images:
   - `cv2.putText`
   - `cv2.circle`
   - `cv2.rectangle`
   - `cv2.line`
4. Edge/Corner detection:
   - `cv2.goodFeaturesToTrack`
   - `cv2.Canny`

5. perspective transform:
   - `cv2.drawContours` 
   - `cv2.getPerspectiveTransform`
   - `cv2.warpPerspective`
